Overview dashboard built into GitHub showing CI/CD performance metrics for an organization.
Covers average job run time, queue time, job failure rate, and breakdowns by repository,
workflow, job, runner type, and OS.

Useful for tracking CI reliability and identifying bottlenecks in the build pipeline.

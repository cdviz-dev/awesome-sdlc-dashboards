Overview dashboard built into GitHub showing CI/CD usage metrics for an organization.
Covers total minutes consumed, job runs, and breakdowns by repository, workflow, job, runner type, and OS.

Useful for tracking CI costs and identifying which repositories or workflows consume the most build time.
